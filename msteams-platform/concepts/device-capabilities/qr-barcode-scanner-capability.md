## <a name="code-snippet"></a><span data-ttu-id="67802-203">Trecho de código</span><span class="sxs-lookup"><span data-stu-id="67802-203">Code snippet</span></span>

<span data-ttu-id="67802-204">**Chamada `ScanBarCode()` API** para verificação de QR ou código de barras usando câmera:</span><span class="sxs-lookup"><span data-stu-id="67802-204">**Calling `ScanBarCode()` API** for scanning QR or barcode using camera:</span></span>

```javascript
const config: microsoftTeams.media.BarCodeConfig = {
  timeOutIntervalInSec: 30};
microsoftTeams.media.scanBarCode((error: microsoftTeams.SdkError, decodedText: string) => {
  if (error) {
    if (error.message) {
      output(" ErrorCode: " + error.errorCode + error.message);
    } else {
      output(" ErrorCode: " + error.errorCode);
    }
  } else if (decodedText) {
    output(decodedText);
  }
}, config);
```
