# <a name="app-templates-for-microsoft-teams"></a>Modelos de aplicativos para o Teams

Os modelos de aplicativo são exemplos de aplicativos completos para Microsoft Teams que são de software livre e estão disponíveis GitHub. Cada modelo de aplicativo contém instruções detalhadas para implantar e instalar esse aplicativo para sua organização. Ele também fornece um aplicativo de exemplo que você pode instalar e começar a usar imediatamente. O código-fonte completo também está disponível, o que permite explorá-lo em detalhes ou bifurcar o código e alterá-lo para atender aos seus requisitos específicos.
Todos os modelos de aplicativo são fornecidos nos termos [de licença do MIT](https://github.com/OfficeDev/microsoft-teams-apps-eprescription/blob/master/LICENSE) .

> [!NOTE]
> Você deve licenciar e dar suporte a aplicativos criados com base em modelos de aplicativo para seus usuários e organizações.

Os principais benefícios do uso de modelos de aplicativo são:

* **Implante diretamente na nuvem:** Todos os modelos de aplicativo incluem scripts de implantação que permitem hospedar todos os serviços necessários no Microsoft Azure ou no Power Platform.
* **Código de exemplo recomendado:** Os modelos de aplicativo estão em conformidade com as práticas recomendadas de segurança e infraestrutura. Todas as alterações enviadas pela comunidade aos modelos de aplicativo são revisadas para garantir a conformidade.
* **Personalizável e extensível:** Embora todos os modelos de aplicativo sejam implantados com configuração mínima, toda a base de código e os scripts de implantação são fornecidos, para que você possa personalizá-los ou estendê-los facilmente para atender às suas necessidades exclusivas.
* **Documentação detalhada:** Todos os modelos de aplicativo são acompanhados pela documentação de ponta a ponta sobre as etapas de arquitetura, implantação e configuração da solução.  

## <a name="adoption-tool--champion-management-platform"></a>Ferramenta de Adoção – Plataforma de Gerenciamento Campeão

O modelo de aplicativo CMP (Champion Management Platform) ajuda você a gerenciar, dimensionar e inspirar seus campeões de trabalho em equipe para obter mais. Esse modelo de aplicativo é criado no Estrutura do SharePoint e carregado em uma guia dentro de uma equipe. Os grupos podem aproveitar essa ferramenta para ajudar a gerenciar a associação ao programa, fornecer um placar de líderes e tipos de eventos para registro em log e ferramentas para sobrepor selos digitais aos participantes do programa.

[Obter em GitHub](https://github.com/OfficeDev/microsoft-teams-apps-champion-management)

## <a name="adoption-tool--microsoft-365-learning-pathways-get-started"></a>Ferramenta de Adoção – Microsoft 365 Learning Pathways (Introdução)

O Introdução modelo de aplicativo permite que você traga o poder Microsoft 365 roteiros de aprendizagem dentro Microsoft Teams. Esse modelo de aplicativo permite que você conceda acesso fácil a páginas de treinamento específicas ou outros ativos da intranet e carregue o conteúdo diretamente no Teams. Você também pode alterar o nome ou logotipo do aplicativo para corresponder à identidade visual da sua empresa.

[Obter em GitHub](https://github.com/msft-teams/tools/tree/master/M365%20Learning%20Pathways)

## <a name="company-communicator"></a>Communicator da empresa

O aplicativo Communicator empresa permite que as equipes corporativas criem e enviem mensagens destinadas a várias equipes ou a um grande número de funcionários por chat, permitindo que a organização alcance os funcionários exatamente onde eles colaboram. Utilize esse modelo para vários cenários, como novos comunicados de iniciativa, integração de funcionários, aprendizado moderno e desenvolvimento ou transmissões em toda a organização.

O aplicativo fornece uma interface fácil para os usuários designados criarem, visualizarem, colaborarem e enviarem mensagens.

Ele fornece uma base para criar recursos personalizados de comunicação direcionada, como telemetria personalizada, sobre quantos usuários reconheceram ou interagiram com uma mensagem.

[Obter em GitHub](https://github.com/OfficeDev/microsoft-teams-company-communicator-app)

## <a name="co-worker-appreciation"></a>Reconhecimento de colegas de trabalho

Usando o modelo de apreciação de colegas de trabalho no Microsoft Teams, os usuários podem reconhecer as conquistas de seus colegas dentro do contexto Teams trabalho. Quando os colegas de trabalho selecionam recompensar um colega, os destinatários e outros membros da equipe são marcados em uma conversa de canal e recebem uma notificação sobre os detalhes do prêmio do canal. Os prêmios são gravados no aplicativo Teams, que é seguro, portátil e facilmente compartilhável. Isso é considerado como a versão baseada no PowerApps do modelo de aplicativo Open Badges, com um placar de líderes.

## <a name="faq-plus"></a>Perguntas Frequentes Plus

As perguntas&um bot são uma maneira fácil de fornecer respostas para perguntas frequentes feitas pelos usuários. No entanto, a maioria dos bots não consegue se envolver com os usuários de maneira significativa porque não há humanos no loop quando o bot falha. O bot de perguntas frequentes é&um bot amigável que coloca um humano em loop quando não consegue ajudar. É possível fazer uma pergunta ao bot e o bot responderá com uma resposta se ele estiver contido no base de dados de conhecimento. Caso contrário, o bot permite que o usuário envie uma consulta que é postada em uma equipe pré-configurada de especialistas que ajudam a fornecer suporte agindo com base nas notificações de dentro da própria equipe.

> [!NOTE]
> A versão mais recente do **FaQ Plus** dá suporte a resoluções de Q&A aprimoradas, permitindo que uma equipe de especialistas conclua o seguinte:
>
> &#x2714; adicionar novas&como diretamente ao base de dados de conhecimento usando extensões de mensagem.
>
> &#x2714; editar e excluir pares Q&A adicionados por um bot.
>
> &#x2714; acompanhar o histórico de revisão de Q&As.
>
> &#x2714; configurar uma resposta com detalhes adicionais a serem exibidos como um [Cartão Adaptável](../task-modules-and-cards/cards/cards-reference.md#adaptive-card).
>
[Obter em GitHub](https://github.com/OfficeDev/microsoft-teams-apps-faqplusv2)

## <a name="icebreaker"></a>Frases de apresentação

Icebreaker é um [bot Microsoft Teams que](../bots/what-are-bots.md) ajuda sua equipe a se aproximar emparelhando dois membros aleatórios da equipe a cada semana para se reunirem. O bot facilita o agendamento sugerindo automaticamente horários gratuitos que funcionam para ambos os membros. Fortaleça conexões pessoais e crie uma comunidade fortemente unida com este aplicativo.

Além de incentivar conexões pessoais em toda a sua equipe, o aplicativo Icebreaker pode ajudar a cultivar comunidades baseadas em interesse em sua organização. Por exemplo, você pode usar esse aplicativo para um grupo de DevOps para ajudar ideias e práticas recomendadas a serem distribuídas organicamente em toda a organização.

[Obter em GitHub](https://github.com/OfficeDev/microsoft-teams-icebreaker-app)
