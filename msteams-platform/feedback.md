# <a name="support-and-feedback"></a>Suporte e comentários

## <a name="product-support-and-service-issues"></a>Problemas de serviço e suporte ao produto

Use o **Centro de Administração do Azure** ou o **Centro de Administração do Microsoft 365** para quaisquer **problemas críticos para os negócios** que precisem de suporte baseado em SLA.

| **Suporte ao produto** | **Contato** |
|:------------|:------------|
| **Problemas e falhas de produtos do Microsoft Teams** | Se você tiver um contrato de suporte Premier para o :::image type="icon" source="assets/icons/microsoft-icon.png" border="false"::: Microsoft 365, visite o Centro de Administração do Microsoft 365 e use o menu Suporte para [abrir uma solicitação de serviço ](https://admin.microsoft.com/). |
| **Ajuda e suporte do Azure** | Se você tiver uma assinatura paga do Azure, visite o Centro de Administração do Azure para [abrir um tíquete](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/newsupportreq). |
| **Perguntas gerais sobre o Microsoft Teams** | Se você tiver perguntas gerais sobre o Microsoft Teams, envie suas perguntas para a :::image type="icon" source="assets/icons/microsoft-icon.png" border="false"::: [Comunidade do Microsoft Teams](https://answers.microsoft.com/msteams/forum).|

## <a name="developer-community-help"></a>Ajuda da comunidade de desenvolvedores

A comunidade de desenvolvedores do Microsoft Teams usar o Stack Overflow e as Perguntas e Respostas da Microsoft para se conectar com outros desenvolvedores para idealizar, obter esclarecimentos e enviar consultas. Além disso, você também pode usar outros contatos ou sites, dependendo do tipo de ajuda da comunidade necessária para levantar problemas, limitações e perguntas gerais.

### <a name="developer-community-forums"></a>Fóruns da comunidade de desenvolvedores

Poste suas perguntas e ajude outros membros da comunidade compartilhando e respondendo às perguntas sobre o Desenvolvimento de Aplicativos do Microsoft Teams.

| **Canal da comunidade**|   **Contato**  |
|:---------------------|:---------------|
|**Ajuda da comunidade** | <ul> <li>:::image type="icon" source="assets/icons/stack-overflow-icon.png" border="false"::: [Stack Overflow](https://stackoverflow.com/questions/tagged/microsoft-teams): Use a marca `microsoft-teams` para postar perguntas. Siga as diretrizes do Stack Overflow, tais como o fornecimento de um título descritivo, uma declaração concisa do problema e detalhes para reproduzir o problema. Solicitações de recursos ou perguntas gerais estão fora do tópico. Se você for um novo usuário, confira a [ Central de Ajuda do Stack Overflow ](https://stackoverflow.com/help).</li>  <li>:::image type="icon" source="assets/icons/microsoft-icon.png" border="false"::: [Perguntas e Respostas da Microsoft](/answers/topics/office-teams-app-dev.html): Amamos o Stack Overflow e fornecemos amplo suporte às perguntas postadas no Stack Overflow. Use a marca `office-teams-app-dev` para postar perguntas. O Stack Overflow tem critérios sobre perguntas apropriadas e as [Perguntas e Respostas da Microsoft](/answers/topics/office-teams-app-dev.html) tem uma política aberta em relação a isto. </li><ul> |
| **Grupo de conversas da comunidade** | Use a sala de chat informal da comunidade para :::image type="icon" source="assets/icons/Teams-icon.png" border="false":::[desenvolvedores de aplicativos do Microsoft Teams](https://gitter.im/OfficeDev/MicrosoftTeamsAppDev). Selecione **ENTRAR PARA COMEÇAR A FALAR**, se estiver entrando pela primeira vez. Selecione **INGRESSAR NA SALA**, se já estiver conectado. |

### <a name="report-issues"></a>Relatar problemas

Envie problemas e faça perguntas gerais relacionadas ao SDK e amostras usando os canais de ajuda da comunidade.

| **Canal da comunidade** | **Contato** |
|:----------------------|:------------|
| **Interrupções da plataforma** | Verifique se seu problema já foi levantado nas [interrupções da plataforma do Teams](https://aka.ms/TeamsPlatform/Outages) antes de levantar um. |
| **Exemplos** | Envie problemas com amostras para o respectivo repositório de [Amostras do Microsoft Teams](https://github.com/OfficeDev/Microsoft-Teams-Samples).|
|  **Problemas de CLI e SDK do TeamsFx ou do Kit de Ferramentas do Teams** | <ul><li> :::image type="icon" source="assets/icons/GitHub-icon.png" border="false":::  [ Problemas do GitHub ](https://github.com/OfficeDev/TeamsFx/issues): Criar  novos problemas no [repositório do GitHub do TeamsFx (Microsoft Teams Framework)](https://github.com/OfficeDev/TeamsFx) para relatar problemas ou levantar solicitação de recursos. Recomendamos o uso de problemas do GitHub para consultas e para receber ajuda da comunidade. <li> :::image type="icon" source="assets/icons/stack-overflow-icon.png" border="false"::: [Stack Overflow](https://stackoverflow.com/questions/tagged/teams-toolkit): Use a marca `teams-toolkit` para postar perguntas. Siga as diretrizes do Stack Overflow, tais como fornecer um título descritivo, uma declaração concisa do problema e detalhes para reproduzir o problema. Solicitações de recursos ou perguntas gerais estão fora do tópico. </li> </ul> |
| **Problemas ou sugestões do SDK de extensão de bot e mensagem** | Envie problemas e solicitações de recursos para o repositório do SDK para a linguagem do seu bot ([C#](https://github.com/Microsoft/botbuilder-dotnet/), [JavaScript](https://github.com/Microsoft/botbuilder-js), ou [Python](https://github.com/Microsoft/botbuilder-python)). Publique perguntas acerca de instruções no [Stack Overflow](https://stackoverflow.com/questions/tagged/botframework%20microsoft-teams) usando as marcas `botframework` e `microsoft-teams`. |
| **Problemas ou sugestões do SDK da guia** | Envie problemas e solicitações de recursos para o repositório do SDK da [Biblioteca JavaScript do Microsoft Teams](https://github.com/OfficeDev/microsoft-teams-library-js/issues). Use a marca `microsoft-teams` para postar perguntas acerca de instruções sobre o SDK no [Stack Overflow](https://stackoverflow.com/questions/tagged/microsoft-teams). |
| **Live Share sugestões ou problemas do SDK** | Envie problemas e solicitações de recursos para o repositório do SDK [para Live Share SDK](https://github.com/microsoft/live-share-sdk). Use a `live-share` marca e `microsoft-teams` a marca para postar perguntas de instruções sobre o SDK no [Stack Overflow](https://stackoverflow.com/questions/tagged/live-share+microsoft-teams). |
